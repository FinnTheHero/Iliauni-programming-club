## Find source code

Source code is always linked at the end of every page.

## Search for a document

You can press **" / "** (Slash) and start typing what you want to search, if the topic exists on the website you will find it.

## Publish a new document

### Easy
Use **Markdown** editor of your choice
1. Edit the text you want to publish
2. Add following lines at the top of the file, change them accordingly
3. Send it to one of our moderators in our discord server.

!!!
Following example is from this page.
!!!
```js
---
label: "How to"
description: "Everything about how to use this website."
icon: note
order: 100
category: How-to
date: 2024-2-7
---
```
You can also check [Retype](https://retype.com) for more information about how to format text, add icons, buttons, tags...

### Hard

You can follow steps in [Git basic guide](/Docs/Git/basics.md) to add or modify documents.<br />
After that, you need to make a pull request. Your changes will be reviewed and merged.

## Fix a mistake in the document
If you have found mistake in the document, go to website's source code, fix the issue and submit a pull request,

Or

Edit the document with issue and send it in our Discord server to one of the moderators.
